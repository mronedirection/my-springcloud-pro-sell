### *什么是日志框架*

- 是一套能实现日志输出的工具包
- 能够描述系统运行状态的所有时间都可以算作是日志

    - 用户下线
    - 接口超时
    - 数据库崩溃

### *日志框架能力*

1. 定制输出目标
2. 定制输出格式
3. 携带上下文信息
4. 运行是选择输出

### *日志级别*

1. ERROR
2. WARN
3. INFO
4. DEBUG
5. TRACE
